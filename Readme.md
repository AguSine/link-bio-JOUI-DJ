## 🚀 Instalación y ejecución

```bash
# Clonar el repositorio
git clone https://github.com/AguSine/link-bio-JOUI-DJ.git
cd link-bio-JOUI-DJ

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar la app
reflex run
